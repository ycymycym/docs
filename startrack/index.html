隐私政策 · Privacy Policy

StarTrack（以下简称"本应用"）非常重视用户隐私。本政策说明我们如何收集、使用和保护您的信息。

1. 我们收集的信息

本应用为家长管理儿童成长记录而设计，不要求提供真实姓名或电子邮件。我们收集以下信息：

  • 匿名账户：通过 Supabase 进行匿名登录，生成唯一的用户 ID，不与真实身份关联。
  • 家庭与孩子信息：由家长主动输入的孩子名字、年龄段、头像（emoji 或照片）。
  • 使用数据：任务完成记录、星星余额、奖励兑换记录、成长打卡（身高体重、照片、视频、语音）。
  • 推送通知令牌：通过 Firebase 获取设备令牌，用于发送提醒通知（可在系统设置中关闭）。
  • 订阅状态：通过 RevenueCat 处理 Pro 订阅，我们仅保存订阅状态，支付信息由 Apple 处理。

2. 如何使用信息

  • 提供 App 功能（任务、奖励、成长记录）
  • 跨设备同步家庭数据
  • 发送您选择开启的推送提醒
  • 验证订阅状态以解锁 Pro 功能

我们不将您的数据用于广告推送、行为分析或向第三方销售。

3. 第三方服务

  • Supabase：数据库与匿名认证，数据加密存储。
  • Firebase（Google）：推送通知服务。
  • RevenueCat：订阅管理，不处理支付卡信息。
  • Apple：处理 App 内购支付。

4. 儿童隐私

本应用由家长使用，儿童不直接使用本服务。孩子的数据由家长掌控，家长可随时删除。我们不向儿童收集个人信息，符合 COPPA 相关要求。

5. 数据安全与删除

所有数据通过 TLS 加密传输，服务器端加密存储。您可随时在 App 设置中删除孩子档案或注销账户，删除后数据不可恢复。

6. 政策修订

本政策如有重大变更，我们将在 App 内通知。继续使用本应用即视为接受更新后的政策。

7. 联系我们

如对本政策有任何疑问，请发送邮件至 flyskyau1@gmail.com

如何在新设备上恢复数据？

在新设备安装 App 后，进入「设置 → 加入已有家庭」，输入旧设备上的家庭邀请码即可恢复所有数据。请妥善保存邀请码。

如何取消 Pro 订阅？

前往 iPhone「设置 → Apple ID → 订阅」，找到 StarTrack 并取消。当前计费周期内仍可使用 Pro 功能。

我的成长照片/视频存在哪里？

媒体文件加密存储在 Supabase 云端服务器，并缓存于本地设备。删除孩子档案会同步删除所有相关媒体。

App 在网络不好时能用吗？

可以。数据在本地 Hive 数据库中缓存，打开 App 立即显示。完成任务、兑换奖励等操作会先写入本地，联网后自动同步云端。

How do I restore data on a new device?

Install the App on your new device, go to Settings → Join Existing Family, and enter the family invite code from your old device. All data will be restored. Please keep your invite code safe.

How do I cancel my Pro subscription?

Go to iPhone Settings → Apple ID → Subscriptions, find StarTrack, and cancel. Pro features remain active until the end of the current billing period.

Where are photos and videos stored?

Media files are encrypted and stored on Supabase cloud servers, with a local device cache. Deleting a child's profile permanently removes all associated media.

Does the App work with a poor connection?

Yes. Data is cached locally in a Hive database so the App opens instantly. Actions like completing tasks or redeeming rewards are written locally first and sync to the cloud when online.

新しいデバイスでデータを復元するには？

新しいデバイスにアプリをインストールし、「設定 → 既存の家族に参加」に進み、旧デバイスの家族招待コードを入力してください。すべてのデータが復元されます。招待コードは大切に保管してください。

Pro サブスクリプションをキャンセルするには？

iPhone の「設定 → Apple ID → サブスクリプション」で StarTrack を見つけてキャンセルしてください。現在の請求期間の終了まで Pro 機能を引き続きご利用いただけます。

写真や動画はどこに保存されますか？

メディアファイルは Supabase クラウドサーバーに暗号化保存され、端末にもキャッシュされます。子どものプロフィールを削除すると、関連するすべてのメディアも完全に削除されます。

電波が悪いときでも使えますか？

はい。データはローカルの Hive データベースにキャッシュされているため、アプリはすぐに起動します。タスク完了や報酬交換などの操作はまずローカルに書き込まれ、オンラインになったときにクラウドと同期します。

새 기기에서 데이터를 복원하려면?

새 기기에 앱을 설치한 후 설정 → 기존 가족에 참여로 이동하여 이전 기기의 가족 초대 코드를 입력하세요. 모든 데이터가 복원됩니다. 초대 코드를 안전하게 보관하세요.

Pro 구독을 취소하려면?

iPhone 설정 → Apple ID → 구독으로 이동하여 StarTrack을 찾아 취소하세요. Pro 기능은 현재 결제 기간이 끝날 때까지 계속 이용할 수 있습니다.

사진과 동영상은 어디에 저장되나요?

미디어 파일은 Supabase 클라우드 서버에 암호화되어 저장되며, 로컬 기기에도 캐시됩니다. 자녀 프로필을 삭제하면 관련 미디어가 모두 영구적으로 삭제됩니다.

네트워크 상태가 나쁠 때도 사용할 수 있나요?

네. 데이터는 로컬 Hive 데이터베이스에 캐시되어 앱이 즉시 열립니다. 과제 완료나 보상 교환 같은 작업은 먼저 로컬에 기록되고 온라인 상태가 되면 클라우드와 동기화됩니다.

Làm thế nào để khôi phục dữ liệu trên thiết bị mới?

Cài đặt Ứng dụng trên thiết bị mới của bạn, đi đến Cài đặt → Tham gia gia đình hiện có, và nhập mã mời gia đình từ thiết bị cũ của bạn. Tất cả dữ liệu sẽ được khôi phục. Hãy giữ mã mời của bạn an toàn.

Làm thế nào để hủy đăng ký Pro?

Vào Cài đặt iPhone → Apple ID → Đăng ký, tìm StarTrack và hủy. Tính năng Pro vẫn hoạt động đến hết chu kỳ thanh toán hiện tại.

Ảnh và video được lưu ở đâu?

Các tệp phương tiện được mã hóa và lưu trữ trên máy chủ đám mây Supabase, với bộ nhớ cache cục bộ trên thiết bị. Xóa hồ sơ trẻ sẽ xóa vĩnh viễn tất cả phương tiện liên quan.

Ứng dụng có hoạt động khi kết nối kém không?

Có. Dữ liệu được lưu vào bộ nhớ cache cục bộ trong cơ sở dữ liệu Hive nên Ứng dụng mở ngay lập tức. Các thao tác như hoàn thành nhiệm vụ hoặc đổi thưởng được ghi cục bộ trước và đồng bộ lên đám mây khi có kết nối.

Bagaimana cara memulihkan data di perangkat baru?

Instal Aplikasi di perangkat baru Anda, buka Pengaturan → Bergabung dengan Keluarga yang Ada, dan masukkan kode undangan keluarga dari perangkat lama Anda. Semua data akan dipulihkan. Harap simpan kode undangan Anda dengan aman.

Bagaimana cara membatalkan langganan Pro?

Buka Pengaturan iPhone → Apple ID → Langganan, temukan StarTrack, dan batalkan. Fitur Pro tetap aktif hingga akhir periode penagihan saat ini.

Di mana foto dan video disimpan?

File media dienkripsi dan disimpan di server cloud Supabase, dengan cache lokal di perangkat. Menghapus profil anak akan menghapus semua media terkait secara permanen.

Apakah Aplikasi berfungsi dengan koneksi buruk?

Ya. Data di-cache secara lokal dalam database Hive sehingga Aplikasi terbuka secara instan. Tindakan seperti menyelesaikan tugas atau menukar hadiah ditulis secara lokal terlebih dahulu dan disinkronkan ke cloud saat online.
