Privacy Policy
简体中文

StarTrack（以下简称"本应用"）非常重视用户隐私。本政策说明我们如何收集、使用和保护您的信息。

1. 我们收集的信息

本应用为家长管理儿童成长记录而设计，不要求提供真实姓名或电子邮件。我们收集以下信息：

  • 匿名账户：通过 Supabase 进行匿名登录，生成唯一的用户 ID，不与真实身份关联。
  • 家庭与孩子信息：由家长主动输入的孩子名字、年龄段、头像（emoji 或照片）。
  • 使用数据：任务完成记录、星星余额、奖励兑换记录、成长打卡（身高体重、照片、视频、语音）。
  • 推送通知令牌：通过 Firebase 获取设备令牌，用于发送提醒通知（可在系统设置中关闭）。
  • 订阅状态：通过 RevenueCat 处理 Pro 订阅，我们仅保存订阅状态，支付信息由 Apple 处理。

2. 如何使用信息

  • 提供 App 功能（任务、奖励、成长记录）
  • 跨设备同步家庭数据
  • 发送您选择开启的推送提醒
  • 验证订阅状态以解锁 Pro 功能

我们不将您的数据用于广告推送、行为分析或向第三方销售。

3. 第三方服务

  • Supabase：数据库与匿名认证，数据加密存储。
  • Firebase（Google）：推送通知服务。
  • RevenueCat：订阅管理，不处理支付卡信息。
  • Apple：处理 App 内购支付。

4. 儿童隐私

本应用由家长使用，儿童不直接使用本服务。孩子的数据（名字、记录）由家长掌控，家长可随时删除。我们不向儿童收集个人信息，符合 COPPA 相关要求。

5. 数据安全与删除

所有数据通过 TLS 加密传输，服务器端加密存储。您可随时在 App 设置中删除孩子档案或注销账户，删除后数据不可恢复。

6. 政策修订

本政策如有重大变更，我们将在 App 内通知。继续使用本应用即视为接受更新后的政策。

7. 联系我们

如对本政策有任何疑问，请发送邮件至 flyskyau1@gmail.com

常见问题 · FAQ

如何在新设备上恢复数据？

在新设备安装 App 后，进入"设置 → 加入已有家庭"，输入旧设备上的家庭邀请码即可恢复所有数据。请妥善保存邀请码。

如何取消 Pro 订阅？

前往 iPhone「设置 → Apple ID → 订阅」，找到 StarTrack 并取消。当前计费周期内仍可使用 Pro 功能。

我的成长照片/视频存在哪里？

媒体文件加密存储在 Supabase 云端服务器，并缓存于本地设备。删除孩子档案会同步删除所有相关媒体。

App 在中国大陆使用会慢吗？

我们已做本地缓存优化：数据在本地 Hive 数据库中缓存，打开 App 立即显示，后台同步云端。网络不佳时操作（打勾任务、兑换奖励）仍会立即响应，联网后自动同步。

How do I restore data on a new device?

Install the App on your new device, go to Settings → Join Existing Family, and enter the family invite code from your old device. All data will be restored. Please keep your invite code safe.

How do I cancel my Pro subscription?

Go to iPhone Settings → Apple ID → Subscriptions, find StarTrack, and cancel. Pro features remain active until the end of the current billing period.

Where are photos and videos stored?

Media files are encrypted and stored on Supabase cloud servers, with a local device cache. Deleting a child's profile permanently removes all associated media.
